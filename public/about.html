About Mixr

Mixr is a web application made for cocktail lovers who want to explore different types of cocktails or even find new ones.

How to Use

Home Page - Search for different cocktail names and ingredients. You can also favorite these results.


My Cocktails Page - Saved Cocktails will be stored on this page for easy access and later use.

About the Data

All cocktail information is provided by the CocktailDB public API.
